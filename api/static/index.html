Système
⚡ Système actif
Active ou met en pause le suivi des prises, les alertes et la collecte de données. Quand le système est en pause, aucune donnée n'est générée.
Modes de fonctionnement
🔄 Mode remplissage
Désactive la détection HX711 pour remplir les compartiments. Ouvrez le couvercle pour naviguer avec les boutons poussoir.
🔔 Buzzer
Active/désactive les alertes sonores de la boîte.
Contextes spéciaux
🌙 Mode Ramadan
Décale les horaires : shour avant le fajr, midi suspendu, iftar après le coucher du soleil.
🏥 Hospitalisation
Suspend complètement le système. Les médicaments sont gérés par l'hôpital.
